### 📌 About Me

I'm a 2nd-year BCA student at **M.G.R. College, Hosur** (Periyar University) and **NSS Programme Officer** on campus. I run **JOJO Digital**, a one-person AI-assisted studio building websites, automation, and creative assets for local businesses that don't have agency budgets.

My pitch model is simple: **build the demo first, then show it** — no proposals, no mockups, just a live working site.

### 💡 What Drives Me

I like solving real, small-business problems with tools that don't cost anything to run — GitHub Pages, Google Apps Script, free-tier AI APIs. If it works without a billing account, I'll find a way to ship it.

**Tech Stack:** HTML, CSS, JavaScript, Google Apps Script, Python, Canva, AI-assisted development
